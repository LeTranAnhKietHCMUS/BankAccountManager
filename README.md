# BankAccountManager

## 🇬🇧 English

A bank account management system developed as an Object-Oriented Programming (OOP) course project at Ho Chi Minh City University of Science. The system is implemented in two parts:

- **C++ (Console):** Core business logic demonstrating OOP principles — encapsulation, inheritance, polymorphism, and abstraction.
- **C# WinForms (GUI):** A graphical interface built on top of the same logic.

### Features
- Customer profile management (create, search, update)
- Account types: Payment, Savings, and Loan accounts
- Transactions: deposit, withdrawal, transfer, transaction history
- Savings account management: interest calculation, maturity rollover, early withdrawal with penalty
- Loan account management: payment schedule, monthly repayment, repayment history
- Revenue reporting and customer statistics
- Data persistence via CSV files

### Technologies
- C++ (OOP: abstract classes, virtual methods, polymorphism)
- C# / .NET WinForms
